# SIAMDA

Sistem Informasi Absensi Mobile Digital berbasis PWA dan Google Apps Script.

## Fitur MVP

- Login User dan Admin
- Dashboard User
- Dashboard Admin
- Absensi Masuk/Pulang
- GPS Radius Lock / Map Lock
- Kamera selfie untuk validasi wajah
- Riwayat absensi
- Backend Google Apps Script
- Database Google Sheets
- PWA-ready
